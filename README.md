# gori-deep-train

An open-source project providing generic deep learning training, evaluation and analysis pipelines based on PyTorch. Read the documentation [here](https://gori-deep-train.readthedocs.io/en/latest/).

<font color="red">**DISCLAIMER**</font>: While this project is fully functional, it is not currently in a stable or production-ready state. Documentation may not be complete, many dependencies may be missing and you may find known and unknown bugs. This repository is mainly for showcase purposes, rather than to provide a fully supported, out-of-the-box framework.
